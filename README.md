# 🗳️ Online Voting System (Prototype)

A secure and user-friendly web-based voting system prototype that allows users to cast votes in real-time. The system ensures fair voting by allowing only one vote per authenticated user.

## 🚀 Features

- 🔐 User Authentication System  
- 🗳️ Real-time Voting Interface  
- 👤 One Vote Per User Logic  
- 📋 Candidate Listing Page  
- ⚡ Fast Backend using Express.js  
- 🎨 Clean and Responsive UI  

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript  
- Node.js  
- Express.js

## ⚙️ Installation & Setup

1. Clone the repository:
git clone https://github.com/your-username/online-voting-system.git

2. Open project folder:
cd online-voting-system

3. Install dependencies:
npm install

4. Start the server:
node server.js

5. Open in browser:
http://localhost:3000
